# -HeartIQ-Disease-Analysis-Dashboard-4th-year-project-part-1
The Heart Disease Analysis Dashboard is a web-based tool that analyzes patient health data (like age, cholesterol, blood pressure, etc.) to detect risk patterns and predict the likelihood of heart disease using data visualization and analytics.
